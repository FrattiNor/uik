<h1 align="center">UIK</h1>

文档位置：https://frattinor.github.io/uik/

### 📦 Install

```
npm install uik
```

```
yarn add uik
```

### 🔨 Usage

```
import { Button, Tooltip } from 'uik';

const App = () => (
    <Tooltip title="notice text">
        <Button type="primary">hover me</Button>
    <Tooltip/>
);
```
