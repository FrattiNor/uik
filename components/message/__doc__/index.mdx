# message 消息提示

message 用于全局展示操作的反馈信息。

## 何时使用

可提供成功、警告和错误等反馈信息。

显示并自动消失，是一种不打断用户操作的轻量级提示方式。

## API

组件提供了一些静态方法，使用方式和参数如下：

-   `message.open(title, options)`
-   `message.config(config)`
-   `message.destory()`

<div className="api">

| 属性  | 说明     | 类型                | 默认值 | 版本 |
| :---- | :------- | :------------------ | :----- | :--- |
| title | 反馈信息 | string \| ReactNode | -      | -    |

</div>

`options` 对象属性如下：

<div className="api">

| 属性      | 说明                                                                           | 类型                                                                          | 默认值    | 版本 |
| :-------- | :----------------------------------------------------------------------------- | :---------------------------------------------------------------------------- | :-------- | :--- |
| type      | 消息类型                                                                       | success \| error \| warn \| info \| default                                   | default   | -    |
| id        | 消息的唯一标识，用于 destory 关闭特定 message                                  | number                                                                        | -         | -    |
| position  | 弹出消息的位置                                                                 | topLeft \| topCenter \| topRight \| bottomLeft \| bottomCenter \| bottomRight | topCenter | -    |
| duration  | 弹出消息显示时长秒数`（设为 0 时不自动关闭）` 覆盖全局 message 配置的 duration | number                                                                        | 3         | -    |
| showClose | 显示关闭按钮                                                                   | boolean                                                                       | false     |
| desc      | 对弹出消息的描述                                                               | string \| ReactNode                                                           | -         | -    |
| width     | 弹出消息的宽度（限长后可能导致 title 换行）                                    | number                                                                        | -         | -    |

</div>

`config` 对象属性如下：

<div className="api">

| 属性        | 说明                                              | 类型    | 默认值   | 版本 |
| :---------- | :------------------------------------------------ | :------ | :------- | :--- |
| maxCount    | 最大弹出消息数量                                  | number  | Infinity | -    |
| duration    | 弹出消息显示时长秒数`（设为 0 时不自动关闭）`     | number  | 3        | -    |
| overAnimate | 超出 maxCount 后的新增的 message 是否需要 animate | boolean | false    | -    |

</div>
