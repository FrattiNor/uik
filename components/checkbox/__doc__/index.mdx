## API

<div className="api">

| 属性           | 说明                      | 类型                       | 默认值 | 版本 |
| :------------- | :------------------------ | :------------------------- | :----- | :--- |
| checked        | checkbox 的值             | boolean                    | -      | -    |
| defaultChecked | checkbox 的默认值         | boolean                    | false  | -    |
| checkedHalf    | 是否选中部分              | boolean                    | -      | -    |
| disabled       | 是否禁用                  | number                     | false  | -    |
| onChange       | checkbox 值发生改变的回调 | (checked: boolean) => void | -      | -    |
| className      | 类名                      | string                     | -      | -    |

</div>

## Checkbox.Group

| 属性               | 说明                          | 类型                             | 默认值 | 版本 |
| :----------------- | :---------------------------- | :------------------------------- | :----- | :--- |
| options            | checkbox 列表                 | string[] \| option[]             | []     | -    |
| checkedList        | checkboxList 的值             | boolean[]                        | -      | -    |
| defaultCheckedList | checkboxList 的默认值         | boolean[]                        | []     | -    |
| disabled           | 是否全部禁用                  | boolean                          | false  | -    |
| onChange           | checkboxList 值发生改变的回调 | (checkedList: boolean[]) => void | -      | -    |
| className          | 类名                          | string                           | -      | -    |

## option

```
type checkboxGroupOption = {
  label: string;
  value: string;
  disabled?: boolean;
}
```
