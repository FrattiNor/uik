## API

<div className="api">

| 属性           | 说明                   | 类型                       | 默认值 | 版本 |
| :------------- | :--------------------- | :------------------------- | :----- | :--- |
| checked        | Switch 的值            | string                     | -      | -    |
| defaultChecked | Switch 的默认值        | string                     | false  | -    |
| disabled       | 是否禁用               | boolean                    | false  | -    |
| onChange       | 输入框内容变化时的回调 | (checked: boolean) => void | -      | -    |

</div>
