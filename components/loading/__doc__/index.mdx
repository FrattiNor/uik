# Loading 加载中

用于页面和区块的加载中状态。

## 何时使用

页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。

## API

<!-- 给table设置外部className 下方需要留空一行（不然mdx解析会出错） -->

`Loading` 属性如下：

<div className="api">

| 属性      | 说明             | 类型    | 默认值 | 版本 |
| :-------- | :--------------- | :------ | :----- | :--- |
| loading   | 是否为加载中状态 | boolean | false  | -    |
| inline    | 是否为行内元素   | boolean | false  | -    |
| className | 类属性           | string  | -      | -    |

</div>
