# Loading 加载中

用于页面和区块的加载中状态。

## 何时使用

用于等待异步数据或正在渲染过程时。

## 代码演示

<CodeWrapper>
    <CodeBox
        title="基础"
        getComponent={() => import('./demo-basic')}
        getDesc={() => import('./demo-basic/desc.md')}
        getJs={() => import('!raw-loader!./demo-basic')}
        getLess={() => import('!raw-loader!./demo-basic/index.less')}
    />
    <CodeBox
        title="行内元素Loading"
        getComponent={() => import('./demo-inline')}
        getDesc={() => import('./demo-inline/desc.md')}
        getJs={() => import('!raw-loader!./demo-inline')}
        getLess={() => import('!raw-loader!./demo-inline/index.less')}
    />
</CodeWrapper>

## API

<!-- 给table设置外部className 下方需要留空一行（不然mdx解析会出错） -->

`Loading` 属性如下：

<div className="api">

| 属性      | 说明             | 类型    | 默认值 | 版本 |
| :-------- | :--------------- | :------ | :----- | :--- |
| loading   | 是否为加载中状态 | boolean | false  | -    |
| inline    | 是否为行内元素   | boolean | false  | -    |
| className | 类属性           | string  | -      | -    |

</div>

`LoadingIcon` 属性如下：

<div className="api">

| 属性      | 说明   | 类型   | 默认值 | 版本 |
| :-------- | :----- | :----- | :----- | :--- |
| className | 类属性 | string | -      | -    |

</div>
