# Sticky 固钉

将页面元素钉在可视范围。

## 何时使用

在较长的滚动页面中固定导航条在顶部等。

## API

<div className="api">

| 属性         | 说明                                                   | 类型                      | 默认值 | 版本 |
| :----------- | :----------------------------------------------------- | :------------------------ | :----- | :--- |
| offsetTop    | 距离滚动窗口底部达到指定偏移量后触发                   | number                    | -      | -    |
| offsetBottom | 距离滚动窗口顶部达到指定偏移量后触发                   | number                    | -      | -    |
| rootId       | 设置滚动窗口的 id（不设置默认为视窗）                  | string                    | -      | -    |
| getRoot      | 获取滚动窗口（用于 ref 获取 dom）（优先级高于 rootId） | () => HTMLElement \| null | -      | -    |
| style        | 设置元素的样式                                         | CSSProperties             | -      | -    |
| className    | 设置元素的样式                                         | string                    | -      | -    |

</div>

## 注意

保证元素在滚动窗口可见
