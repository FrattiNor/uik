## API

<div className="api">

| 属性          | 说明                   | 类型                     | 默认值   | 版本 |
| :------------ | :--------------------- | :----------------------- | :------- | :--- |
| value         | input 的值             | string                   | -        | -    |
| defaultValue  | input 的默认值         | string                   | 空字符串 | -    |
| allowClear    | 是否显示一键清除       | boolean                  | false    | -    |
| disabled      | 是否禁用               | boolean                  | false    | -    |
| maxLength     | input 的最大输入长度   | number                   | -        | -    |
| size          | 设置 input 的大小      | large \| middle \| small | middle   | -    |
| onValueChange | 输入框内容变化时的回调 | (value: string) => void  | -        | -    |
| onEnter       | 按下回车的回调         | (value: string) => void  | -        | -    |
| placeholder   | input 未输入时的提示   | string                   | -        | -    |
| width         | 手动设置 input 宽度    | number                   | -        | -    |

</div>
