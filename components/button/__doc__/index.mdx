# Button 按钮

承载点击事件。

## 何时使用

响应用户点击行为。

在 UIK 中提供了三种样式。

-   默认按钮：白板按钮。
-   主题按钮：主题色按钮（主要操作）。
-   危险按钮：红色按钮（危险操作）。

以及两种状态属性。

-   禁用：操作不可用。
-   加载中：用于长时间的点击事件的反馈。

## API

<!-- 给table设置外部className 下方需要留空一行（不然mdx解析会出错） -->

<div className="api">

| 属性      | 说明                                               | 类型                         | 默认值  | 版本 |
| :-------- | :------------------------------------------------- | :--------------------------- | :------ | :--- |
| htmlType  | 设置 button 原生的 type 值，可选值请参考 HTML 标准 | string                       | button  | -    |
| className | 类属性                                             | string                       | -       | -    |
| disabled  | 设置按钮禁用                                       | boolean                      | false   | -    |
| loading   | 设置按钮加载中状态                                 | boolean                      | false   | -    |
| size      | 设置按钮大小                                       | large \| middle \| small     | middle  | -    |
| type      | 设置按钮类型                                       | primary \| danger \| default | default | -    |
| onClick   | 点击按钮时的回调                                   | (event) => void              | -       | -    |

</div>

支持原生 button 的其他所有属性。
