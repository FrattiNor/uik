# Button 按钮

按钮用于点击事件开始一个即时操作。

## 何时使用

标记了一个操作命令，响应用户点击行为，触发相应的事件。

在 UIK 中提供了三种样式按钮。

-   默认按钮：用于没有主次之分的操作。
-   主题按钮：用于主要操作。
-   危险按钮：删除/移动/修改权限等危险操作，一般需要二次确认。

以及两种状态属性与上面配合使用。

-   禁用：操作不可用的时候，一般需要文案解释。
-   加载中：用于异步操作等待反馈的时候，也可以避免多次提交。

## API

通过设置 Button 的属性来产生不同的按钮样式，推荐顺序为：`type` -> `shape` -> `size` -> `loading` -> `disabled`。

按钮的属性说明如下：

<!-- 给table设置外部className 下方需要留空一行（不然mdx解析会出错） -->

<div className="api">

| 属性      | 说明                                               | 类型                         | 默认值  | 版本 |
| :-------- | :------------------------------------------------- | :--------------------------- | :------ | :--- |
| htmlType  | 设置 button 原生的 type 值，可选值请参考 HTML 标准 | string                       | button  | -    |
| className | 类属性                                             | string                       | -       | -    |
| disabled  | 设置按钮禁用                                       | boolean                      | false   | -    |
| loading   | 设置按钮加载中状态                                 | boolean                      | false   | -    |
| size      | 设置按钮大小                                       | large \| middle \| small     | middle  | -    |
| type      | 设置按钮类型                                       | primary \| danger \| default | default | -    |
| onClick   | 点击按钮时的回调                                   | (event) => void              | -       | -    |

</div>

支持原生 button 的其他所有属性。
