## 共同的 API

<div className="api">

| 属性            | 说明                                   | 类型                                     | 默认值                    | 版本 |
| :-------------- | :------------------------------------- | :--------------------------------------- | :------------------------ | :--- |
| format          | DatePicker 识别的 string 格式          | string                                   | YYYY-MM-DD                | -    |
| disabledDate    | 设置 DatePicker 的禁用区域             | (currentDay: Dayjs \| string) => boolean | -                         | -    |
| allowClear      | 是否显示一键清除                       | boolean                                  | true                      | -    |
| disabled        | 是否禁用                               | boolean                                  | false                     | -    |
| size            | 设置 DatePicker 的大小                 | large \| middle \| small                 | middle                    | -    |
| valueType       | 设置 DatePicker 的进入和输出的数据类型 | string \| Dayjs                          | Dayjs \| 根据输入类型判断 | -    |
| textBefore      | 显示在前面的 text 字段                 | string                                   | -                         | -    |
| visible         | 窗口是否显示                           | boolean                                  | -                         | -    |
| onVisibleChange | visible 发生改变的回调                 | (visible: boolean) => void               | -                         | -    |
| topDom          | 弹窗顶部添加元素，用于添加自定义功能   | ReactElement                             | -                         | -    |
| bottomDom       | 弹窗低部添加元素，用于添加自定义功能   | ReactElement                             | -                         | -    |
| leftDom         | 弹窗左部添加元素，用于添加自定义功能   | ReactElement                             | -                         | -    |
| rightDom        | 弹窗右部添加元素，用于添加自定义功能   | ReactElement                             | -                         | -    |

</div>

## DatePicker

<div className="api">

| 属性         | 说明                            | 类型                | 默认值 | 版本 |
| :----------- | :------------------------------ | :------------------ | :----- | :--- |
| value        | DatePicker 的值                 | date                | -      | -    |
| defaultValue | DatePicker 的默认值             | date                | null   | -    |
| onChange     | DatePicker 的值发生改变时的回调 | (day: date) => void | -      | -    |

</div>

## RangPicker

<div className="api">

| 属性         | 说明                            | 类型                        | 默认值       | 版本 |
| :----------- | :------------------------------ | :-------------------------- | :----------- | :--- |
| value        | RangPicker 的值                 | [date, date]                | -            | -    |
| defaultValue | RangPicker 的默认值             | [date, date]                | [null, null] | -    |
| onChange     | RangPicker 的值发生改变时的回调 | (day: [date, date]) => void | -            | -    |

</div>
