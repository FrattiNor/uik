## API

<div className="api">

| 属性           | 说明                             | 类型                       | 默认值 | 版本 |
| :------------- | :------------------------------- | :------------------------- | :----- | :--- |
| checked        | checkbox 的值                    | boolean                    | -      | -    |
| defaultChecked | checkbox 的默认值                | boolean                    | false  | -    |
| checkedHalf    | 是否选中部分                     | boolean                    | -      | -    |
| disabled       | 是否禁用                         | number                     | false  | -    |
| onChange       | checkbox 值发生改变的回调        | (checked: boolean) => void | -      | -    |
| className      | 类名                             | string                     | -      | -    |
| value          | 用判断于 Checkbox.Group 选中的值 | string \| number           | index  | -    |

</div>

## Checkbox.Group

| 属性               | 说明                          | 类型                                                | 默认值 | 版本 |
| :----------------- | :---------------------------- | :-------------------------------------------------- | :----- | :--- |
| checkedList        | checkboxList 的值             | (string \| number) []                               | -      | -    |
| defaultCheckedList | checkboxList 的默认值         | (string \| number) []                               | []     | -    |
| disabled           | 是否全部禁用                  | boolean                                             | false  | -    |
| onChange           | checkboxList 值发生改变的回调 | (checkedList: (string \| number) []) => void        | -      | -    |
| onChange           | checkboxList 值发生改变的回调 | (checkedAll: boolean, checkedHalf: boolean) => void | -      | -    |
| className          | 类名                          | string                                              | -      | -    |
