## API

<div className="api">

| 属性        | 说明             | 类型    | 默认值 | 版本 |
| :---------- | :--------------- | :------ | :----- | :--- |
| name        | 语义化的图标名称 | string  | -      | -    |
| className   | 类属性           | string  | -      | -    |
| defaultIcon | 使用默认的图标库 | boolean | false  | -    |

</div>

组件提供 `config` 方法，配置 iconfont，对象属性如下：

<div className="api">

| 属性   | 说明                                    | 类型   | 默认值 | 版本 |
| :----- | :-------------------------------------- | :----- | :----- | :--- |
| prefix | iconfont 的 icon-name 的前缀            | string | -      | -    |
| url    | iconfont 的 js 链接（可不填，自己引入） | string | -      | -    |

</div>

## 注意

请确保导入的 `iconfont.js` 文件的 `<svg>` 元素不存在 `fill` 属性，不然会导致设置的 `color` 样式失效。
建议使用正则匹配 `/fill="#([0-9]|[a-z]|[A-Z])+"/` 替换为空字符串
