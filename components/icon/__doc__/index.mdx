## API

<div className="api">

| 属性      | 说明             | 类型   | 默认值 | 版本 |
| :-------- | :--------------- | :----- | :----- | :--- |
| name      | 语义化的图标名称 | string | -      | -    |
| className | 类属性           | string | -      | -    |

</div>


组件提供 `config` 方法，配置 iconfont，对象属性如下：

<div className="api">

| 属性       | 说明                                     | 类型 | 默认值 | 版本 |
| :--------- | :--------------------------------------- | :--- | :----- | :--- |
| fontFamily | iconfont 的 css 属性 font-family         | -    | -      | -    |
| url        | iconfont 的 css 链接（可不填，自己引入） | -    | -      | -    |

</div>
