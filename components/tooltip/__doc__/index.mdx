# Tooltip 文字提示

文字提示气泡框。

## 何时使用

鼠标移入则显示提示，移出消失，承载普通文本。

## 代码演示

<CodeWrapper>
    <CodeBox title="基本" desc={descNormal} code={codeNormal}>
        <DemoNormal />
    </CodeBox>
    <CodeBox title="定位" desc={descPosition} code={codePosition}>
        <DemoPosition />
    </CodeBox>
</CodeWrapper>

## 通用 API

<div className="api">

| 属性                  | 说明                                                     | 类型                                                                                                                                                           | 默认值    | 版本 |
| :-------------------- | :------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------- | :-------- | :--- |
| position              | 定位                                                     | topLeft \| topCenter \| topRight \| bottomLeft \| bottomCenter \| bottomRight \| leftTop \| leftCenter \| leftBottom \| rightTop \| rightCenter \| rightBottom | topCenter | -    |
| visible               | 是否可见                                                 | boolean                                                                                                                                                        | false     | -    |
| containerZIndex       | 容器的 zIndex， 用于和其他悬浮内容比较层次               | number                                                                                                                                                         | -         | -    |
| rootId                | 挂载容器的 id                                            | string                                                                                                                                                         | body      | -    |
| getRoot               | 获取挂载的容器（用于 ref 获取 dom）（优先级高于 rootId） | () => HTMLElement \| null                                                                                                                                      | body      | -    |
| trigger               | 不设置 visible 时，设置触发 visible 的方式               | hover \| focus \| click                                                                                                                                        | hover     | -    |
| autoAdjust            | 自动调整 Tooltip 位置不被遮挡                            | boolean                                                                                                                                                        | false     | -    |
| disabled              | 阻止 trigger 触发 visible 修改                           | boolean                                                                                                                                                        | false     | -    |
| onVisibleChange       | trigger 触发的 visible 改变的回调                        | (visible) => void                                                                                                                                              | -         | -    |
| popSameWidth          | 弹出层最小宽度和触发元素的宽度一致                       | boolean                                                                                                                                                        | false     | -    |
| overlayStyle          | 弹出层的样式                                             | CSSProperties                                                                                                                                                  | -         | -    |
| overlayClass          | 弹出层的样式                                             | string                                                                                                                                                         | -         | -    |
| updatePositionDepends | 触发重新获取定位的依赖属性                               | any[]                                                                                                                                                          | -         | -    |

</div>

## 注意

请确保 `子元素` 能接受 `onMouseEnter`、`onMouseLeave`、`onFocus`、`onBlur`, `onClick` 事件。

请确保 `挂载容器` 的定位存在（不存在定位，不会跟随滑动）

`挂载容器` 确定后不可更改
